

# BLE Devices Demo

 

## Summary
- **Summary:**  Android BLE Devices Demo
- **Created Date:** 23 July 2023
- **Last modified:** 23 July 2023 

## Feature
- Search Bluetooth
- List all near bluetooth devices
- Connect
- See Characteristics (Information)



### Technology Used
- Kotlin
- Java
- Bluetooth
- RecyclerView
- Card View
- Constraint Layout
- Runtime Permissions

### Backend Details
